### RestTemplate / RestClient : 단발성 통신 & 전통적인 동기(JPA) 환경
- 외부 API 호출이 서비스 로직 내에서 가끔 발생하고, 응답을 무조건 기다렸다가 DB에 저장해야 할 때 씁니다.

### WebClient : 비동기 스트리밍 & 대용량/오래 걸리는 통신
- 외부 API 호출이 서비스 로직 내에서 가끔 발생하고, 응답을 무조건 기다렸다가 DB에 저장해야 할 때 씁니다.

### OpenFeign : MSA 환경 & 여러 외부 API를 밥 먹듯이 호출할 때
- 데이터를 쪼개서 받거나, 타 서버 응답이 10초 이상 걸려서 내 서버 스레드를 놀리지 않고 딴일 시켜야 할 때(Non-blocking) 씁니다.
